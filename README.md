# Signature DocTech 🚀

A web-based, production-ready Clinical Prescription Workstation designed for medical professionals to generate, structure, and export pixel-perfect, printable clinical records securely.

👉 **Live Application:** [https://prescription-app-psi.vercel.app](https://prescription-app-psi.vercel.app)

## 🛠️ Tech Stack & Architecture

* **Frontend Framework:** Next.js (React)
* **Styling & Layout:** Tailwind CSS
* **Backend Database:** Supabase (PostgreSQL)
* **Hosting & Deployment:** Vercel

---

## ⚡ Core Engineering Highlights

* **Dual-Role Authentication Gateway:** Engineered a secure workspace gate that utilizes environment variables for real medical professional roles while gracefully handling an isolated sandbox context for public reviewers.
* **Async Database Query Lookup:** Coupled the application to a Supabase database instance allowing dynamic medicine and generic indexing optimized with custom debounce thresholds to avoid network flooding.
* **Dynamic A4 Document Matrix:** Built a live-updating, strict-bound CSS rendering sheet modeled explicitly around real A4 paper dimensions for uniform print and PDF workflows.
* **Contextual Medical Guidelines:** Integrated pre-formatted responsive instruction matrices natively in Bengali for orthopedic and trauma specialties (Knee, Back, Plaster recovery, and Physical Therapy).
